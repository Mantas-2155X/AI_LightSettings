# AI_LightSettings

This plugin allows you to change some light settings.

**Installation**  
* Drop AI_LightSettings.dll into `BepInEx/plugins/`.

**Configuration**  
* Found in the F1 menu under AI_LightSettings category. 

**Features:**  
* Backlight Intensity.  

Left: Full intensity  
Right: Zero intensity
